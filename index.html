Shortcut iOS — capture rapide
Version dictée + catégorisée par Claude (recommandé)

Ton idée est dictée, Claude la catégorise (task / note / idea / meeting) et elle apparaît dans la section Captures d'Aujourd'hui pour triage en 2 taps.

  1. App Raccourcis iOS → + Nouveau → nomme Focus Capture
  2. Dicter le texte (langue : français, arrêt : pause)
  3. Obtenir le contenu de l'URL — appel Claude
    URL : https://api.anthropic.com/v1/messages
    Méthode : POST
    En-têtes : x-api-key: [ta clé sk-ant-…], anthropic-version: 2023-06-01, content-type: application/json
    Corps (JSON) :
    {"model":"claude-haiku-4-5-20251001","max_tokens":200,"system":"Tu catégorises une note vocale. Types: task (action concrète avec verbe), note (info à stocker), idea (intuition à creuser), meeting (rdv/appel). Réponds en JSON strict: {\"kind\":\"task|note|idea|meeting\",\"title\":\"...<60c\",\"context\":\"...<80c\"}.","messages":[{"role":"user","content":"[Texte dicté]"}]}
  4. Obtenir la valeur du dictionnaire : content.1.text
  5. Obtenir le contenu de l'URL — lecture du Gist
    URL : https://api.github.com/gists/[gistId], Méthode : GET
    En-têtes : Authorization: Bearer [token], Accept: application/vnd.github+json
  6. Obtenir la valeur du dictionnaire : files.capture-inbox.json.content — si l'action échoue, utilise {"items":[]}
  7. Texte JavaScript (action « Exécuter JavaScript sur page web » ou construction manuelle) : construire le JSON final :
    {"items":[...anciens items, {"id":"cap-[timestamp]","timestamp":"[now ISO]","raw":"[Texte dicté]",...parse Claude text...}]}
  8. Obtenir le contenu de l'URL — écriture Gist
    URL : idem, Méthode : PATCH
    + en-tête Content-Type: application/json
    Corps : {"files":{"capture-inbox.json":{"content":"[JSON final]"}}}
  9. Ajoute à l'écran d'accueil · déclenche via Siri « Capture Focus »
